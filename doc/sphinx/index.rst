*******
Purpose
*******

The ICOstate package contains a stateful application interface for the `ICOtronic system`_.

.. _ICOtronic system: https://www.mytoolit.com/icotronic/

************
Requirements
************

- `Python 3`_

.. _Python 3: https://www.python.org

*******
Install
*******

.. code-block:: shell

   pip install icostate
